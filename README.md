# ⚡ ST-Info-Blocks

Мощное и аккуратное расширение для SillyTavern, позволяющее автоматически генерировать и встраивать кастомные информационные блоки (статус сцены, инвентарь, погода, скрытые мысли персонажа) прямо в сообщения чата на основе контекста ролевой игры.

## ✨ Особенности

* **Нативная интеграция в чат:** Больше никаких "висящих" поверх экрана элементов, исчезающих при перезагрузке! HTML-код инфоблоков аккуратно встраивается прямо в текст сообщения бота. ST сохраняет их навсегда, а при необходимости вы можете отредактировать их вручную через стандартную кнопку редактирования (карандашик).
* **Умная группировка (Экономия токенов):** Вы можете объединять несколько разных блоков в одну группу. Расширение автоматически склеит их промпты и отправит *одним* запросом к нейросети.

📦 Пример работы: один профиль — два запроса
Профиль (подключения API) содержит всё:
🔹 Статус
🔹 Инвентарь
🔹 Комментарии
🔹 Картинка
🧩 Группа 1 (текстовая) ➜ Статус + Инвентарь + Комментарии 📨 Отправляется единым сообщением
🖼️ Группа 2 (медийная) ➜ Картинка (и только она)
🔁 Общий процесс
Профиль 1
    ├─ Запрос 1 → текстовая группа (статус+инвентарь+комментарии)
    └─ Запрос 2 → медиа-группа (картинка)
Вы можете делать сколько угодно групп с одним профилем подключения API и комбинировать запросы по разному.

* **Организация по папкам:** Настроили десятки инфоблоков под разные сюжеты? Разложите их по папкам (категориям). Удобные сворачивающиеся списки с красивой анимацией не дадут меню настроек превратиться в хаос.

* **Чистый и приятный визуал:** Никакого визуального мусора. Во время генерации отображаются аккуратные спиннеры загрузки. Если API выдаст ошибку — она появится в виде системного уведомления (toast) в углу экрана, не засоряя ваш лог чата красным текстом.
* **Защита от спама запросами:** Скрипт распознает уже сгенерированные блоки. Случайное редактирование сообщения или свайп на старый ответ не запустят генерацию по второму кругу.
* **Гибкие настройки:** Отключайте генерацию при свайпах для конкретных блоков, настраивайте температуру, количество токенов и длину контекста индивидуально под каждую задачу.

## ❓ FAQ (Частые вопросы)

**Q: Как объединить несколько блоков в один запрос, чтобы сэкономить токены?** **A:** В настройках каждого блока есть поле **«ID Группы»**. Просто впишите туда одинаковое слово (например, `group1` или `survival`) для нужных блоков. Главное, чтобы у них был выбран одинаковый профиль API. Расширение само отправит их одной пачкой!

**Q: Как быстро открыть настройки расширения?** **A:** В верхней панели SillyTavern (там же, где настройки пользователя и расширений) добавлена удобная иконка папок. Один клик — и меню инфоблоков открыто.

**Q: Как создать новую папку для блоков?** **A:** В редакторе блока в поле **«Папка (Категория)»** просто впишите любое название. Если папки с таким именем еще нет, она создастся автоматически. Ранее созданные папки будут доступны в выпадающем списке этого же поля.

## 🛠️ Установка / Обновление

1. Зайдите в **Extensions** -> **Install Extension**.
2. Используйте ссылку: `https://github.com/Gromova-Olga/ST-Info-Blocks`
3. Если расширение уже установлено — нажмите **Update** во вкладке расширений.
